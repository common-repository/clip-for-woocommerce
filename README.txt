== Installation ==

Requisitos técnicos mínimos

Para instalar este plugin deberás contar con:

* PHP 7.4 o superior	
* WordPress 5.0 o superior
* WooCommerce 4.0 o superior
* Contar con una cuenta en Clip

Comienza en unos pocos minutos

Puedes instalar el plugin de manera automática o manual:

Manera automática

1. Desde el administrador de WordPress, selecciona “Plugins”-->”Añadir nuevo”.
2. En el buscador de Plugins escribe “Clip” y presiona Enter
3. Selecciona nuestro Plugin y da click en “Instalar ahora”.
4. Una vez instalado serás redirigido a la página de Onboarding para configurar tu cuenta de Clip.
5. Realiza las configuraciones del plugin y guárdalas.
¡Listo!

Manera Manual

1. Descarga el archivo .zip en esta misma página en el botón “descargar” o encuentralo en nuestra documentación.
2. Ingresa al administrador de WordPress selecciona “Plugins”-->”Añadir nuevo”.
3. En la parte superior selecciona “Subir Plugin”
4. Selecciona el archivo .zip guardado previamente en tu computadora y selecciona el botón “Instalar ahora”.
5. Luego al dar click en “Activar” serás redirigido a la página de Onboarding para configurar tu cuenta de Clip.
6. Finalmente dirígete a la sección de “Woocommerce”-->”Ajustes” y realiza las configuraciones que desees.
¡Listo!
